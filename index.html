Welcome to your selectors exercise!

Please write your css in style.css, place it in css folder and link it here, avoid using inline css or <style> block
Complete these tasks:
  1. Set the background of all the headers (1-6) to #222222
  2. Make the word "header" in the second header red
  3. set a black solid border of 1px to every <p> located inside a <div>
  4. make all the articles inside a <section> have green background and a 3px dashed red border
  5. make all the articles inside an <aside> have a yellow background and a 3px dotted red border
  6. make the <aside> background to have this values: red=123, green=0, blue=30. make it half transparent
  7. make the <span> inside the <article> with class "leading" bold. make sure only it is bold and not the <span> inside the <p>
  8. make the last paragraph font to be "Pavanam". use google fonts to find the font and install it
  9. Challange: make the last <p> to be font to be Alef. (google to find it).
Notes:
  1. Write only css, don't change the html
  2. try to experiment with different selectors, there are a few different ways to achieve the required result
Good luck!
